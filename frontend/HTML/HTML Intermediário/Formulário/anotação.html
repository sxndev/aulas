Formulário

todo formulário é composto pela tag *form, ela tem abertura e fechamento.

essa tag contém 2 atributos: *method e *action

Method

o valor do method vai indicar como o formulário será enviado, existem 2 valores: *GET e *POST

GET

o GET serve para "pegar uma informação" a barra de pesquisa do google é um formulário GET

POST

o post serve para POSTar um formulário ao seu destino serve para eu enviar uma informação. a tela de login do facebook é um exemplo de formulário POST

para permitir que o usuário coloque uma foto nesse formulário, temos que usar o seguinte atributo: enctype="multparti/form-data" assim, o formulário estará pronto para enviar arquivos de imagens, se ele não for colocado, o input file não funcionará!
