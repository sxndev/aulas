Input de texto

Todo tipo de tag input deve ter um atributo "type" ( tipo do input)

    types para input text:
  • Name - para inserir nomes
  • Email - para inserir email
  • Number - para inserir números
  • Password - para digitar senha
  • Submit - cria um botão e envia para um formulário
  • Checkbox - cria uma caixa para seleção
  • Radio - Cria uma bolinha de seleção única, diferente do checkbox que pode ter vários itens selecionados
  • Button - Cria um botão
  • File - Permite que o usuário adicione um arquivo

    • Um atributo importante, é o *placeholder, o valor dele aparecerá "no fundo" do elemento do imput, isso facilitará o usuário na hora de saber qual campo ele está preenchendo, Exemplo
