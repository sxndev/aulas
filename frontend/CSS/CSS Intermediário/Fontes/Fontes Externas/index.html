Fontes Externas

Para usar uma fonte externa, é necessário baixá-la e colocá-la dentro de uma pasta, não precisa de todos os arquivos dentro dela, para o funcionamento correto, é preciso que na pasta tenha somente o arquivo .otf, .ttf ou um dos dois.

para estilizar no css, precisa escrever o seguinte comando @font-face,dentro dele terá um font-family onde deve ser colocado o nome da fonte, e no url é só buscar a fonte na pasta

Detalhe importante

nos parenteses e no título da fonte, deve conter aspas!
