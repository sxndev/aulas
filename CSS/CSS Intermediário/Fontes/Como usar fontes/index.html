Como selecionar a fonte no CSS

Exemplo:

font-family: Arial, Helvetica, sans-serif;

a primeira fonte a ser usada será a Arial, se por algum motivo o dispositivo não conseguir carregá-la, ela tentara a outra e se caso ela também não carregue, uma fonte do mesmo estilo será escolhida parecida (o estilo da fonte é o nome que aparece depois da segunda vírgula)
