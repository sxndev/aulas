Usando fontes no google fonts

Para utilizar uma ou mais fontes do google fontes basta seguir os seguintes passos:

  1. Escolher uma fonte
  2. Copiar o código dela
  3. Colar na tag "head"
  4. Usar o comando
    font-family=(nome da fonte escolhida)
