Tags semanticas:

As tags a seguir ajudaram as ferramentas de busca a "varrerem" nosso site de uma forma melhor e deixará nosso site com uma maior relevância

  • Header - serve para formar um cabeçário ou agrupar títulos e etc (pode ter mais de uma tag *header na página) ela não precisa ser somente usada para ser a header de um site, mas ela também pode ser usada na tag section

  • Nav geralmente usa-se para fazer o menu do site

  • Main - é onde vamos colocar o conteúdo principal da página, de maior relevância ( pode usar mais de um por página, mas não é recomendado )

  • Section - representa seções dentro da página e podem ter cabeçarios dentro delas, e as tags dentro delas devem estar relacionadas

  • Footer - tag usada para fazer o rodapé de uma página, também pode ser usado para fazer o rodapé de uma section, nele pode conter: contato, redes sociais, saiba mais, política de privacidade e etc

  • Article - serve para escrever conteúdos que sejam independentes, tipo artigos de blogs. é recomendado que cada article tenha uma tag de título e pode conter vários articles dentro de uma página, sem problemnas

  • Aside - serve para colocar conteúdos adicionais (links, noticias, anuncios e etc)

  • figure serve para adicionar uma legenda a nossa imagem, e dentro dela deve ter a tag *figcaption

  • Address serve para definir coisas como telefone, endereço, email, redes sociais e geralmente pode ser colocada na tag *footer
