Links internos

o link interno nada mais é do que um link que leva para otra página do meu site, ao invés de me levar para outro site como link externo faz, ele me leva para outra página do mesmo site.

a estrutura de um link interno é a mesma que a de um link comum,usaremos uma pasta de contato como exemplo:

contato

no valor do href, devo colocar a página html que eu quero, (o uso do atributo target fica a meu critério, mas é melhor não usar)

e para voltar para a página inicial, basta colocar a tag do link e o valor do atributo "href" deve ser "../arquivo da página inicial" sem o atributo "target"
