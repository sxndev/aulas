Links Externos

Para criar ou colocar um link em uma página html, basta usar a tag a, porém essa tag precisa de um atributo,este atributo é o href="*coloque seu link aqui".

Se eu quero que quando cliquem no link a minha página não feche, eu devo usar outro atributo nessa tag, esse atributo é o target=_"blank"

O formato do código deve ficar assim:

*a href="seu link dentro das aspas"**

E se voçe deseja que outra aba seja aberta quando o usuário clique no link, basta adicionar o atributo, e o formato do link ficará assim:

*a href="seu link"* target_"blank"*
